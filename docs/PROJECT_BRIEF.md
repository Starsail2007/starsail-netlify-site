# Project Brief

## 当前项目是什么

这是星帆 Starsail 的个人网站入口项目。它目前承担的是“安静入口”和“待探索空间”的角色：保留基本身份标识、主题切换、文字动效、头像展示和公网部署能力。

## 当前页面状态

- 主标题：待探索的空间
- 说明文案：表达这里目前仅供测试，之后放什么仍待探索
- 默认主题：夜间
- 可切换主题：夜间 / 日间
- 文字动效模式：Calm / Float / Shine / Focus
- 头像交互：点击右上角头像后弹出放大展示
- 公网地址：`https://starsail.netlify.app/`
- GitHub Pages 静态备用地址：`https://starsail2007.github.io/starsail-netlify-site/`

## 为什么整理成这个结构

项目已经从单个 HTML 文件变成可持续开发的静态站点骨架。这样之后无论是改文案、改排版、接入 Figma/Canva 的设计稿，还是逐步加入新的页面与功能，都可以在清晰的目录里进行。

## 给未来 Codex 的提醒

这个项目的“未来要做什么”由用户决定。Codex 应负责把用户的设想转化为可维护的网页结构、组件、样式和交互，不要主动替用户设定内容方向。
